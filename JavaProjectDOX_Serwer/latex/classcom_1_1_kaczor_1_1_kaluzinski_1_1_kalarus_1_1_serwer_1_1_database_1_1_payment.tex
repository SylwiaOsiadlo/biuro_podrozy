\doxysubsection{Detailed Description}
Klasa używana przez framework Hibernate. Służy do tworzenia tabeli przechowywującej dane na temat płatności klientów biura. Tworzy takie encje jak\+: id płatności, id umowy, typ płatności, stan płatności, całkowity koszt. Zawiera także sekwencję automatycznie inkrementującą id płatności.
