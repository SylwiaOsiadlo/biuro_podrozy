\doxysubsection{Detailed Description}
Klasa używana przez framework Hibernate. Służy do tworzenia tabeli przechowywującej dane adresowe klientów aplikacji (pracowników) oraz klientów biura. Tworzy encje id adresu, kraj, miejscowość, numer domu, kod pocztowy oraz ulica. Zawiera sekwencję automatycznie generującą id adresów.
