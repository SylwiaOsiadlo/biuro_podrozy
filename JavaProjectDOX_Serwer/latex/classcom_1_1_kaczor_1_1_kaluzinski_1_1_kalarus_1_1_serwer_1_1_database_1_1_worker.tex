\doxysubsection{Detailed Description}
Klasa używana przez framework Hibernate. Służy do tworzenia tabeli przechowywującej dane dotyczące pracowników biur podróży. Tworzy takie encje jak\+: id pracownika, id biura, id adresu, imie, nazwisko, pesel, login, hasło. Zawiera także sekwencję do automatycznego inkrementowania id pracownika.
