\doxysubsection{Detailed Description}
Klasa używana przez framework Hibernate. Służy do tworzenia tabeli przechowywującej dane na temat wycieczek oferowanych przez biuro podróży. Tworzy encje takie jak\+: id wycieczki, id biura, cena, data wyjazdu, data powrotu, nazwa wycieczki, telefon, maksymalna liczba uczestników, standard. Zawiera także sekwencję automatycznie inkrementującą id wycieczki.
