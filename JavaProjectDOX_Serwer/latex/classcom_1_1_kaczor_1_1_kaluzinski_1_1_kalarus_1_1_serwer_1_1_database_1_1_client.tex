\doxysubsection{Detailed Description}
Klasa używana przez framework Hibernate. Służy do tworzenia tabeli przechowywującej dane na temat klientów biura. Tworzy takie encje jak\+: id klienta, imie, nazwisko pesel, nr telefonu, id adresu, id biura. Zawiera sekwencję automatycznie inkrementującą id klienta.
